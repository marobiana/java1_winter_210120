## 문제) if-elif-else문

### 1. 음수 양수 판별
* 정수를 입력 받고 음수인지 양수인지 출력하세요.
 
> 입력 예시

```
정수를 입력 하세요 : -5
```

> 출력 예시

```
음수입니다
```


### 2. 학점 출력
* 점수를 입력 받고 90 점이상이면 A , 80점 이상 B, 70점 이상 C, 60점 이상 D, 그외 F 를 출력하세요


> 입력 예시

```
점수를 입력 하세요 : 86
```

> 출력 예시

```
B
```

### 3. 비만도 구하기

* BMI 수치를 입력 받아서 비만 판정을 출력하세요.   

	|bmi 수치 |비만도|
	|:---:|:---:| 
	|~ 10 이하|정상 |
	|~ 20 이하| 과체중 |
	|20 ~ 초과| 비만|

> 입력 예시

```
bmi 수치를 입력 하세요 : 18
```
> 출력 예시

```
과체중
```
