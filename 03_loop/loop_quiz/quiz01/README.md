## 문제) while문

### 1. 반복 출력
* 수를 입력 받아서 그 수만큼 "화이팅!!!" 를 출력하세요

> 입력 예시

```
횟수를 입력하세요 : 4
```
> 출력 예시

```
화이팅!!!
화이팅!!!
화이팅!!!
화이팅!!!
```


### 2. 카운트 다운 "발사"
* 수를 입력 받고 그 수에서 부터 0까지 한 줄씩 카운트 다운을 출력 하고 마지막에 "발사"를 출력하세요.

> 입력 예시

```
카운트 다운 수를 입력하세요 : 3
```
> 출력 예시

```
3
2
1
0
발사 
```


### 3. 반복 입력
* while을 이용하여 5번 동안 수를 입력 받고 출력하세요.

> 입출력 예시

```
수를 입력하세요:  : 4
출력 : 4
수를 입력하세요:  : 5
출력 : 5
수를 입력하세요:  : 6
출력 : 6

출력 : 7

출력 : 8
```
