### 4. 등차수열
> 1 4 7 10 13 16 19 22 25 ... 은  
> 1부터 시작해 이전에 만든 수에 3을 더해 다음 수를 만든 수열이다.  
> 이러한 것을 수학에서는 앞뒤 수들의 차이가 같다고 하여
> 등차(차이가 같다의 한문 말) 수열이라고 한다.

* 시작값(a), 등차(d), 몇 번째인지를 나타내는 정수(n)가 입력될 때 n번째 수를 출력하세요.

> 입력 예시1

```
세 개의 수를 입력하세요 : 1 3 10
```
> 출력 예시1

```
28
```

> 입력 예시2

```
세 개의 수를 입력하세요 : 6 4 5
```
> 출력 예시2

```
22
```
