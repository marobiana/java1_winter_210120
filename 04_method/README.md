## 메소드의 기본 구조
- 리턴타입: 결과값의 자료형
```
리턴타입 메소드명(매개변수) {
    // 코드 구현...
    return 결과값;
}
```
> 매개변수(Parameter)가 없는 경우  
```
리턴타입 메소드명() {
    // 코드 구현...
    return 결과값;
}
```
  
> 리턴값이 없는 경우  
```
void 메소드명() {
    // 코드 구현...
}
```
  
> 매개변수(Parameter)가 2개 있는 경우  
```
리턴타입 메소드명(자료형 a, 자료형 b) {
    // 코드 구현...
    return 결과값;
}
```
